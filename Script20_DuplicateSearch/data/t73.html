                Search Cases Help
            

            

                The Search Cases Text box is used to search cases using following fields:



                Intake Case #:This is a unique PVI case number assigned by system to the each record. Searching using this field will list all the unique records matching to the PVI Case number entered by the user



                Event(s): This is a event information which is entered on the case entry screen in Reaction(Coded) or Reaction(Reported) field. Searching using this field will display all the records matching to the Event(S) information entered by the user.



                Product(s): This is a Product information which is entered on the case entry screen on the Product Name field. Searching using this field will display all the records matching to the Product information entered by the user.



                Study Number: This is a Study number which is entered on the case entry screen. Searching using this field will display all the records matching to Study Number information entered by the user.



                Assigned To: This is a user information to whom case is Assigned to. Searching on this field will display all the records which are assigned to that user.
